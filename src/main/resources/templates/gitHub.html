Git Set-up

May 10, 2019

When you set up a new machine, you only need to configure the author once.
If you run "git config -l" you'll see the existing configurations.
Run the following if you're doing this for the first time:
    git config --global user.name "yourUsername"
    git config --global user.email yourEmail@domain.com
In command prompt, go to the project folder, and once you're inside,
initialize the repository and add the corresponding git repository address:
    git init
    git remote add origin https://github.com/yourUsername/yourRepository.git
When you want to save and commit your changes, use "add" and "commit" -- this is like CTRL-S in word doc.
Then use "push" to upload the changes to your Github repository
    git add . git commit -m "commit message"
    git push -u origin master
